Politique de confidentialité

Comment nous collectons, utilisons et protégeons vos données personnelles

Dernière mise à jour : 28 juillet 2025

1. Introduction

Looptra s'engage à protéger votre vie privée et vos données personnelles. Cette politique de confidentialité explique comment nous collectons, utilisons, partageons et protégeons vos informations lorsque vous utilisez nos agents IA spécialisés.

En utilisant nos services, vous acceptez les pratiques décrites dans cette politique.

2. Données que nous collectons

Nous collectons différents types d'informations :

  • Informations d'compte : Email, nom, préférences utilisateur
  • Données d'utilisation : Interactions avec nos agents, historiques de conversations
  • Données métier : Informations partagées pour personnaliser nos agents (itinéraires, préférences de transport, tâches, contextes business)
  • Données techniques : Adresse IP, type de navigateur, données de performance
  • Données d'intégration : Informations récupérées via les APIs connectées (SNCF, Trainline, Notion, etc.) avec votre autorisation
  • Données contextuelles : Informations stockées dans notre base de données sécurisée pour enrichir vos interactions avec nos agents

3. Comment nous utilisons vos données

Vos données sont utilisées pour :

  • Fournir et améliorer nos agents IA spécialisés
  • Personnaliser les réponses et recommandations
  • Maintenir la sécurité et prévenir les abus
  • Communiquer avec vous sur nos services
  • Analyser l'utilisation pour améliorer nos produits
  • Respecter nos obligations légales

Architecture de données : Nous construisons sur l'infrastructure OpenAI/ChatGPT ET nous maintenons notre propre base de données sécurisée. Cette architecture hybride nous permet d'enrichir vos interactions avec du contexte personnalisé tout en respectant les standards de sécurité les plus élevés. Vos données contextuelles sont stockées de manière chiffrée dans nos systèmes européens.

4. Partage des données

Nous ne vendons jamais vos données personnelles. Nous pouvons les partager uniquement dans ces cas :

  • Prestataires de services : OpenAI (ChatGPT), fournisseurs d'APIs (SNCF, Trainline, etc.)
  • Obligations légales : Si requis par la loi
  • Protection des droits : Pour protéger nos droits, votre sécurité ou celle d'autres utilisateurs
  • Avec votre consentement : Dans tous les autres cas, uniquement avec votre accord explicite

5. Sécurité des données

Nous mettons en place des mesures de sécurité appropriées pour protéger vos données dans notre infrastructure :

  • Chiffrement avancé : Données chiffrées en transit (TLS 1.3) et au repos (AES-256)
  • Infrastructure européenne : Nos serveurs sont hébergés en Europe avec des partenaires certifiés
  • Contrôles d'accès stricts : Authentification multi-facteurs et principe du moindre privilège
  • Surveillance continue : Monitoring 24/7 et détection d'anomalies
  • Sauvegardes sécurisées : Copies chiffrées avec rétention géographiquement distribuée
  • Audits réguliers : Évaluations de sécurité par des tiers indépendants
  • Formation de l'équipe : Sensibilisation continue aux bonnes pratiques de sécurité

Nos mesures de sécurité respectent les standards industrie (ISO 27001) et les exigences RGPD.

6. Vos droits

Conformément au RGPD, vous disposez des droits suivants :

  • Accès : Obtenir une copie de vos données personnelles
  • Rectification : Corriger des données inexactes
  • Effacement : Demander la suppression de vos données
  • Limitation : Restreindre le traitement de vos données
  • Portabilité : Récupérer vos données dans un format lisible
  • Opposition : Vous opposer au traitement de vos données
  • Retrait du consentement : Retirer votre consentement à tout moment

Pour exercer ces droits, contactez-nous à privacy@looptra.ai

7. Cookies et technologies similaires

Nous utilisons des cookies pour :

  • Maintenir votre session de connexion
  • Mémoriser vos préférences
  • Analyser l'utilisation de notre site
  • Améliorer votre expérience utilisateur

Vous pouvez configurer votre navigateur pour refuser les cookies, mais cela peut affecter le fonctionnement de nos services.

8. Conservation des données

Nous conservons vos données selon les catégories suivantes :

  • Données de compte : Tant que votre compte est actif
  • Données contextuelles : Stockées dans notre BDD sécurisée pour améliorer vos interactions, supprimées à la fermeture du compte ou sur demande
  • Historiques d'utilisation : Conservés 2 ans maximum pour l'amélioration du service
  • Données de facturation : 10 ans conformément aux obligations légales
  • Logs techniques : 6 mois maximum pour la sécurité et le support

Suppression automatique : Toutes vos données contextuelles stockées dans notre infrastructure sont automatiquement supprimées 3 mois après la fermeture de votre compte, sauf obligation légale contraire.

9. Transferts internationaux

Notre infrastructure : Vos données contextuelles sont stockées exclusivement sur des serveurs européens (France/Allemagne) respectant le RGPD.

Partenaires technologiques : Nos services utilisant l'infrastructure OpenAI, certaines données peuvent être transférées vers les États-Unis lors de vos interactions avec ChatGPT. Ces transferts sont effectués conformément aux mécanismes de protection appropriés :

  • Clauses contractuelles types de la Commission européenne
  • Mesures de sécurité supplémentaires (chiffrement renforcé)
  • Minimisation des données transférées
  • Conformité aux décisions d'adéquation en vigueur

10. Modifications de cette politique

Nous nous réservons le droit de modifier cette politique de confidentialité. En cas de changements importants, nous vous en informerons par email ou via notre site web au moins 30 jours avant l'entrée en vigueur.
